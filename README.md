# ISC2310 - Introduction to Computational Thinking in Data Science with Python

## Course Description

This repository contains materials, assignments, and resources for ISC2310: An Introduction to Computational Thinking in Data Science with Python. The course introduces fundamental concepts of computational thinking and their applications in data science using Python programming language.

## Course Objectives

Students will learn to:
- Understand fundamental computational thinking concepts
- Develop Python programming skills for data science applications
- Work with data structures and algorithms
- Analyze and visualize data using Python libraries
- Apply problem-solving techniques to real-world data science problems
- Implement basic machine learning concepts

## Prerequisites

- Basic understanding of mathematics and logic
- No prior programming experience required
- Eagerness to learn and problem-solve

## Getting Started

### Installation

1. Install Python (version 3.8 or higher) from [python.org](https://www.python.org/downloads/)
2. Install Jupyter Notebook:
   ```bash
   pip install jupyter
   ```
3. Clone this repository:
   ```bash
   git clone https://github.com/mjshooshtari/ISC2310.git
   cd ISC2310
   ```

### Running Jupyter Notebooks

```bash
jupyter notebook
```

## Required Libraries

Common Python libraries used in this course:
- NumPy - Numerical computing
- Pandas - Data manipulation and analysis
- Matplotlib - Data visualization
- Seaborn - Statistical data visualization
- Scikit-learn - Machine learning

Install all required libraries:
```bash
pip install numpy pandas matplotlib seaborn scikit-learn
```

## Course Topics

1. **Introduction to Python Programming**
   - Variables, data types, and operators
   - Control structures and loops
   - Functions and modules

2. **Data Structures**
   - Lists, tuples, and dictionaries
   - Sets and strings
   - File I/O operations

3. **Computational Thinking**
   - Problem decomposition
   - Pattern recognition
   - Abstraction and algorithms

4. **Data Science Fundamentals**
   - Data collection and cleaning
   - Exploratory data analysis
   - Statistical analysis

5. **Data Visualization**
   - Creating effective visualizations
   - Interactive plots
   - Best practices in data presentation

6. **Introduction to Machine Learning**
   - Supervised learning basics
   - Classification and regression
   - Model evaluation
